# Ticketing de Incidencias LAB-LIS  
Sistema de gestión de incidencias para el Laboratorio de Cómputo (LAB-LIS)  

## Descripción 

El sistema Ticketing de Incidencias LAB-LIS permite registrar, clasificar, asignar y gestionar incidencias técnicas de los equipos del laboratorio  

## Objetivo

- Registrar incidencias reportadas por usuarios del laboratorio.
- Clasificar incidencias por tipo (Hardware, Software, Red).
- Asignar prioridad (Baja, Media, Alta, Crítica).
- Asignar técnico responsable.
- Cambiar estado (Pendiente, En Progreso, Terminada).
- Registrar historial de cambios y comentarios.
- Consultar detalles de cualquier incidencia.
- Gestionar equipos y técnicos.
- Mostrar historial completo de la incidencia.
